# Verdant-Website
# 🥗 Healthy Salad Landing Page

A modern and responsive landing page for a healthy salad brand, built using **HTML, CSS, and JavaScript**. The website features a clean design, engaging visuals, smooth interactions, and a user-friendly interface to promote healthy food products.

## 🚀 Live Demo

👉 **Click here to view:**  
**https://sanjana951.github.io/Verdant-Website/**

## ✨ Features

- 🥗 Modern and elegant UI
- 📱 Fully responsive design
- 🌙 Light/Dark mode toggle
- 🛒 Shopping cart icon
- 💰 Product pricing and discount section
- 🍃 Clean and minimal layout
- ⚡ Smooth animations and interactions
- 🌐 Cross-browser compatible

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
